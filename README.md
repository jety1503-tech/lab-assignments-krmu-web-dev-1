Responsive Portfolio Website
Description
A sleek, one-page portfolio website showcasing personal info, skills, projects, and a contact form. Built using HTML and CSS with Flexbox, Grid, media queries, and simple animations for a clean, interactive, and fully responsive experience.​

Viewing Responsive Design
Desktop (≥1024px): Hero text and image side-by-side. Skills and Projects display in grid cards.

Tablet (768px–1023px): Layout adjusts, cards may stack in two columns, sections reflow.

Mobile (≤767px): All content stacks in a single column, hero text appears above the profile image. Use browser resize or device emulation tools to test responsiveness.​

This README is compact, clear, and gives reviewers all essential information and tested breakpoints.
